1.如果key或者value为空抛出异常
2.计算key的hash值
3.死循环
4.如果当前接未初始化就先初始化.再次循环
5.如果当前节点为null，直接通过cas进行把新的node添加到这个位置上，跳出
6.如果当前正在扩容调用helpTransfer，再次循环
7.以要添加元素的node节点为锁进行操作
8.如果node的hash大于0就是个链表，就进行遍历插入到最后一个节点之后
9.否则就是个红黑树就加入红黑树
10.如果binCount不为0就判断一下binCount是不是达到了8，如果到了8就转成红黑树，否则返回Value
11.对当前容量大小进行检查，如果超过了临界值（实际大小*加载因子）就需要扩容
